# 修改UI介面清單3：降低下方轉錄區與底色對比

## 1. 目標
降低簡報頁面底色（暖白）與下方轉錄區顯示區塊之間的視覺對比，使閱讀更柔和、長時間觀看更舒適。

## 2. 影響範圍
- 樣式檔：`css/style.css`
- 頁面：`teacher.html`（簡報模式與一般模式下的轉錄顯示）

## 3. 調整項目
- `.main-column .presentation-subtitle-panel`：背景色微調為更淡的暖灰、加入細邊線。
- `.live-transcription-display`：背景與邊框改為更柔和的暖米色；分隔線改為虛線、降低存在感。
- `.live-transcription-latest`：背景改為淡暖橘、文字改為柔和咖色、取消粗體。
- `.time-tag`：背景與邊框改為更淡的暖色、文字色微調。

## 4. 色彩（Solarized 暖系基調）
- 背景基底：`#fdf6e3`
- 淺暖灰：`#f7f1e4`
- 柔和邊框：`#eadfc9`
- 淡暖橘：`#faecd8`
- 主要文字：`#586e75`
- 強調文字（柔和咖）：`#8a5a2b`

## 5. 驗收標準
1. 轉錄區不再出現過深或突兀的色塊，與頁面底色和諧。
2. 最新語音區的強調仍清晰，但不刺眼；整體可讀性提升。
3. 在 `http://localhost:8000/teacher.html` 預覽無排版異常，色彩一致。

## 6. 回滾方案
若偏好更高對比，可將上述色彩恢復至清單2的設定（背景 `#eee8d5`、最新區文字 `#cb4b16` 等）。
