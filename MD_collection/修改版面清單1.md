# 修改版面清單1：放大即時轉錄輸出的文字\n\n## 目的\n- 提升即時轉錄區（最新語音、已提交轉錄）文字的可讀性，將字級「調大一點」。\n\n## 受影響區域\n- 教師端頁面 `teacher.html` 下方的即時轉錄面板：\n  - 最新語音顯示（`#liveTranscription_latest` / `.live-transcription-latest`）\n  - 已提交轉錄列表（`#liveTranscription_presentation` 包在 `.live-transcription-display` 中）\n  - 時間標籤（`.time-tag`）\n\n## 調整項目（CSS）\n- `.live-transcription-display`：\n  - 新增 `font-size: 1.1em;`、`line-height: 1.6;`\n- `.live-transcription-latest`：\n  - 新增 `font-size: 1.1em;`、`line-height: 1.6;`\n- `.time-tag`：\n  - 由 `12px` 調整為 `13px`\n\n## 變更理由\n- 目前字級偏小，遠距教學或投影時可讀性不足。適度放大字級與行高，提升閱讀舒適度，同時維持既有版面高度設定。\n\n## 驗收標準\n- 最新語音與已提交轉錄的文字明顯變大，行距更舒適。\n- 在簡報模式下（`.container.presentation-mode`）仍能容納預期的行數，無明顯溢出或截斷。\n- 時間標籤與文字比例協調，無需額外捲動即可快速瀏覽。\n\n## 風險與注意\n- 字級放大可能使同一高度顯示的行數略減；若後續需要，可適度上調 `.live-transcription-display` 的 `max-height`。\n- 若裝置解析度較低，可能需要進一步調整左/右欄比例，以維持可讀性。\n\n## 回滾方案\n- 將上述 CSS 調整恢復到原字級與行高（移除新增屬性、還原 `.time-tag` 為 `12px`）。\n
